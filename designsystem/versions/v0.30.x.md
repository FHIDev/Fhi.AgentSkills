## Different vs latest

### fhi-button

- `icon-only`-attributtet er **støttet og ikke deprecated** i v0.30.x.
  (Deprecated i v0.31+.)

```html
<!-- Gyldig i v0.30.x: -->
<fhi-button icon-only variant="subtle" color="neutral">
  <fhi-icon-search></fhi-icon-search>
</fhi-button>
```

## Migrering (0.30 → 0.31)

- Fjern `icon-only`-attributtet fra `fhi-button`; bruk ikon som eneste child:

```html
<!-- Fra: -->
<fhi-button icon-only><fhi-icon-search></fhi-icon-search></fhi-button>

<!-- Til: -->
<fhi-button><fhi-icon-search></fhi-icon-search></fhi-button>
```

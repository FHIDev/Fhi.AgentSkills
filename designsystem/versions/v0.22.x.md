# Delta: v0.22.x

## Scope

Gjelder v0.22.x (alle patcher). **Eldste støttede versjon.**
Skrevet mot: v0.31.0. Innhold er kumulativt — gyldige avvik vs. enhver nyere latest.

## Missing vs latest

- **`fhi-modal-dialog`** — introdusert i v0.29.0, finnes **ikke** i v0.22.x.
  Alternativ: bygg modal manuelt med `<dialog>`-elementet + design tokens.
- **`fhi-body`**, **`fhi-display`**, **`fhi-headline`**, **`fhi-label`**, **`fhi-tag`**, **`fhi-title`** — finnes **ikke** i v0.22.x (verifisert mot npm-manifest).

## Different vs latest

### fhi-button

- `icon-only`-attributtet er **støttet og ikke deprecated** i v0.22.x.
  (Deprecated i v0.31+.)

```html
<!-- Gyldig i v0.22.x: -->
<fhi-button icon-only variant="subtle" color="neutral">
  <fhi-icon-search></fhi-icon-search>
</fhi-button>
```

## Migrering (0.22 → 0.31)

1. Vurder stegvis oppgradering (0.22 → 0.25 → 0.28 → 0.31) for å redusere risiko.
2. Oppgrader til minst v0.29 for å få tilgang til `fhi-modal-dialog`.
3. Fjern `icon-only`-attributtet fra `fhi-button` (deprecated i v0.31):

```html
<!-- Fra: -->
<fhi-button icon-only><fhi-icon-search></fhi-icon-search></fhi-button>

<!-- Til: -->
<fhi-button><fhi-icon-search></fhi-icon-search></fhi-button>
```

4. Test focus/blur-events og width-oppførsel (endret i 0.30.x).
5. Sjekk release notes for alle mellomliggende versjoner for breaking changes.

## Anbefaling

Vurder sterkt oppgradering til latest. Den store avstanden mellom versjonene
øker risikoen for ukjente inkompatibiliteter som ikke er fanget opp i denne deltaen.

## Merk

Denne deltaen er basert utelukkende på release notes og arvet kunnskap
fra nyere delta-filer. Fullstendig verifisering mot npm tarball er ikke utført.
Komponenter og API-endringer mellom 0.22 og 0.27 er ikke kartlagt.

# Delta: v0.36.x

## Scope

Gjelder v0.36.x (alle patcher).
Skrevet mot: v0.40.0. Innhold er kumulativt — gyldige avvik vs. enhver nyere latest.

## Missing vs latest

### Data Table

- Mangler hele komponentgruppen `fhi-data-table`, `fhi-data-table-row` og
  `fhi-data-table-cell`. Den ble innført i v0.37.0. I v0.36.x må tabeller bygges
  med native HTML eller egne komponenter.

### fhi-tag

- Mangler `variant="bordered"` (innført i v0.40.0).

### Ikoner

- Mangler `fhi-icon-file-text` (innført i v0.38.0).

## Different vs latest

### Typografikomponenter (fhi-body, fhi-display, fhi-headline, fhi-label, fhi-title)

- Default `color` er `var(--fhi-color-neutral-text-default)`, **ikke** `currentcolor`
  (currentcolor innført i v0.39.0).

## Legacy-only

Ingen.

## Import/entrypoints

`fhi-text-input` har slottene `start`/`end` (innført v0.35.0). Data Table-entrypoints
finnes ikke.

## Theme/tokens

Ingen endringer. `theme/default.css` er byte-identisk med v0.40.0.

## Rammeverk-notater (Blazor)

Ingen avvik fra latest.

## Patch notes med API-impact

Ingen kjente patches med API-endringer (kun v0.36.0 publisert i 0.36-serien).

## Migrering (0.36 → 0.40)

- Bakoverkompatibel oppgradering.
- Data Table-komponentene blir tilgjengelige fra v0.37.0.
- `fhi-icon-file-text` fra v0.38.0, typografi-`currentcolor` fra v0.39.0,
  `fhi-tag variant="bordered"` fra v0.40.0.

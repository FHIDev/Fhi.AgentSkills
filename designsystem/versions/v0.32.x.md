# Delta: v0.32.x

## Scope

Gjelder v0.32.x (alle patcher).
Skrevet mot: v0.33.0. Innhold er kumulativt — gyldige avvik vs. enhver nyere latest.

## Missing vs latest

- **`fhi-text-input`** mangler slottene `start` og `end` for ikon-plassering
  (innført i v0.35.0). Se [`v0.34.x.md`](v0.34.x.md) for detaljer og runtime-atferd.
- **Data Table** (`fhi-data-table`, `fhi-data-table-row`, `fhi-data-table-cell`)
  mangler — komponentgruppen ble innført i v0.37.0.
- **`fhi-tag`** mangler `variant="bordered"` (innført i v0.40.0).
- **`fhi-icon-file-text`** mangler (innført i v0.38.0).

## Different vs latest

### fhi-grid

- `rows`-property mangler i v0.32.x. Denne ble lagt til i v0.33.0 og setter
  `grid-template-rows: repeat(N, 1fr)`. I v0.32.x brukes standard CSS grid-rader
  (auto-genererte rader basert på innhold).

### Typografikomponenter — default color

- Default `color` er `var(--fhi-color-neutral-text-default)`, **ikke** `currentcolor`.
  Latest endret default til `currentcolor` (arver tekstfarge fra forelder) i v0.39.0.

## Legacy-only

Ingen.

## Import/entrypoints

Runtime-importstier er uendret. Publisert pakke i v0.32.x mangler eksplisitte
`exports` i `package.json` og `.d.ts`-filer per entrypoint — begge innført i
v0.34.0. Konsekvens: svakere IDE-støtte for subpath-importer og modulgjenkjenning.
`custom-elements.json` og `web-types.json` finnes fortsatt i pakken, så HTML- og
custom-element-metadata er tilgjengelig.

## Migrering (0.32 → 0.33)

- Hvis du bruker `fhi-grid` og trenger eksplisitte rader, oppgrader til 0.33.0 for
  den nye `rows`-propertyen. Ellers er oppgraderingen bakoverkompatibel.

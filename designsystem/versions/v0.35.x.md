# Delta: v0.35.x

## Scope

Gjelder v0.35.x (alle patcher).
Skrevet mot: v0.40.0. Innhold er kumulativt — gyldige avvik vs. enhver nyere latest.

## Missing vs latest

### Data Table

- Mangler hele komponentgruppen `fhi-data-table`, `fhi-data-table-row` og
  `fhi-data-table-cell` (innført i v0.37.0).

### fhi-tag

- Mangler `variant="bordered"` (innført i v0.40.0).

### Ikoner

- Mangler `fhi-icon-file-text` (innført i v0.38.0).

## Different vs latest

### Typografikomponenter (fhi-body, fhi-display, fhi-headline, fhi-label, fhi-title)

- Default `color` er `var(--fhi-color-neutral-text-default)`, **ikke** `currentcolor`
  (currentcolor innført i v0.39.0).

## Legacy-only

Ingen.

## Import/entrypoints

`fhi-text-input` har slottene `start`/`end` (innført v0.35.0). `exports`-entrier og
`.d.ts` per entrypoint er på plass (fra v0.34.0). Data Table-entrypoints finnes ikke.

## Theme/tokens

Ingen endringer. `theme/default.css` er byte-identisk med v0.40.0.

## Rammeverk-notater (Blazor)

Ingen avvik fra latest.

## Patch notes med API-impact

Ingen kjente patches med API-endringer (kun v0.35.0 publisert i 0.35-serien).

## Migrering (0.35 → 0.40)

- Bakoverkompatibel oppgradering.
- Data Table fra v0.37.0, `fhi-icon-file-text` fra v0.38.0, typografi-`currentcolor`
  fra v0.39.0, `fhi-tag variant="bordered"` fra v0.40.0.

  publiserte tarballs, typografi-runtime-JS, manifests

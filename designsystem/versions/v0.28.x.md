Skrevet mot: v0.31.0. Innhold er kumulativt — gyldige avvik vs. enhver nyere latest.

## Missing vs latest

- **`fhi-modal-dialog`** — introdusert i v0.29.0, finnes **ikke** i v0.28.x.
  Alternativ: bygg modal manuelt med `<dialog>`-elementet + design tokens.

## Different vs latest

### fhi-button

- `icon-only`-attributtet er **støttet og ikke deprecated** i v0.28.x.
  (Deprecated i v0.31+.)

```html
<!-- Gyldig i v0.28.x: -->
<fhi-button icon-only variant="subtle" color="neutral">
  <fhi-icon-search></fhi-icon-search>
</fhi-button>
```

## Migrering (0.28 → 0.31)

1. Oppgrader til minst v0.29 for å få tilgang til `fhi-modal-dialog`.
2. Fjern `icon-only`-attributtet fra `fhi-button` (deprecated i v0.31):

```html
<!-- Fra: -->
<fhi-button icon-only><fhi-icon-search></fhi-icon-search></fhi-button>

<!-- Til: -->
<fhi-button><fhi-icon-search></fhi-icon-search></fhi-button>
```

3. Test focus/blur-events og width-oppførsel (endret i 0.30.x).
